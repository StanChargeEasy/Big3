Branded action button — geometric Jost label, restrained corners, soft wine-tinted hover lift.

```jsx
<Button variant="primary" size="lg">Book a consultation</Button>
<Button variant="outline" uppercase>Read your chart</Button>
<Button variant="accent" pill>Send message</Button>
```

Variants: `primary` (wine), `accent` (coral), `gold`, `outline`, `ghost`, `inverse` (cream-on-dark).
Sizes: `sm` / `md` / `lg`. Flags: `uppercase`, `pill`, `full`, `disabled`. Slots: `iconLeft`, `iconRight`.
