Small tracked-caps category / status chip (e.g. post categories, tags, counts).

```jsx
<Badge variant="wine">Aries Season</Badge>
<Badge variant="softCoral" size="sm">New Moon</Badge>
<Badge variant="outline">Water Sign</Badge>
```

Variants: solid `wine` / `coral` / `gold` / `ink`; soft washes `softWine` / `softCoral` / `softGold`; `outline`. Sizes `sm` / `md` / `lg`. `pill` defaults true.
